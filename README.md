## About

Digitools is a lightweight web application that puts AI-powered utilities right in your browser. It's designed to be simple to use, fast to load, and easy on the eyes — no clutter, just tools that work.

---

## Features

**⚛️ Modern React 19**  
Built on React 19 with the new `use()` hook and native Suspense support for clean, efficient async rendering and data fetching.

**🎨 Beautiful UI with Tailwind + DaisyUI**  
Every screen is crafted with Tailwind CSS v4 and DaisyUI v5 — consistent, themeable components that look great without extra effort.

**📱 Fully Responsive Design**  
Digitools works seamlessly across all devices. Whether you're on a phone, tablet, or desktop, the layout adapts without breaking a sweat.

**⚡ Fast & Lightweight with Vite**  
Vite 8 keeps the dev experience instant and the production bundle lean. Fast startup, fast builds, fast everything.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/digitools.git
cd digitools

# Install dependencies
npm install

# Run the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Scripts

```bash
npm run dev       # Start development server
npm run build     # Build for production
npm run preview   # Preview production build
npm run lint      # Lint with ESLint
```
